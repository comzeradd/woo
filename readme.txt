== Description ==
The MontyPay Payment Gateway plugin allows you to seamlessly integrate MontyPay's payment options into your WordPress website. It supports three types of integration: S2S, hosted, and wallet integrations.

= Features =
- Securely integrate MontyPay payment options into your website.
- Configure and enable different integration methods: S2S, hosted, and wallets.
- Easily manage merchant keys and passwords provided by the MontyPay team.

== Installation ==
1. Upload the `montypay-payment-gateway` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the plugin settings page and configure your preferred integration method.
4. Enter the merchant key and password provided by the MontyPay team.
5. Enable the MontyPay payment method.

== Frequently Asked Questions ==
= How do I obtain the merchant key and password? =
Contact the MontyPay team to obtain the merchant key and password required for integration.

= Can I use multiple integration methods simultaneously? =
Yes, you can configure and enable multiple integration methods according to your business needs.
